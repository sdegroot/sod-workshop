# zod-workshop

## Installation 

```
npm install
```

## Running tests

### All tests
```
npm run test:all
```

### Specific challenges
```
npm run test:one --challenge=1
```
or 
```
npm run test:one:watch --challenge=1
```

## Challenges

- Defaults
- Coercion
- Type inference
- Custom validations
- custom error messages
- Enums
- Optionals
- Union types 
- Extension types
- Intersection types
- Reuse types
