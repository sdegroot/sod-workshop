# zod-workshop

For all features of Zod, see https://zod.dev/

A high over presentation about Zod: - https://view.genial.ly/64542993eed84100198e6d05/presentation-zod 

## Installation 

```
npm install
```

## Running tests

### All tests
```
npm run test:all
```

### Specific challenges
```
npm run test:one --challenge=1
```
or 
```
npm run test:one:watch --challenge=1
```

## Challenges

- Defaults
- Coercion
- Type inference
- Custom validations
- custom error messages
- Enums
- Optionals
- Union types 
- Extension types
- Intersection types
- Reuse types
